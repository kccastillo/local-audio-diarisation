# Harness Hiccups Register

Append-only log of skill / tool / convention failures observed during agent runs. Newest entries at top.

Each entry:
- **Date:** YYYY-MM-DD
- **Phase / skill:** which phase or skill was running
- **What happened:** silent failure, hard error, deviation, or convention breach
- **Detection:** how it was caught (Ken interrupt, post-review, automated check)
- **Fix / mitigation:** what was changed, or the open question
